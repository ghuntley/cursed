# CURSED v1.0.0-rc1 Installation Guide

## 🚀 Quick Install (Recommended)

### **One-Line Installation**
```bash
curl -sSf https://install.cursedlang.org/v1.0.0-rc1.sh | sh
```

This will:
- Download the appropriate binary for your platform
- Install to `~/.cursed/bin/`
- Add to your PATH
- Verify installation

### **Manual Installation** 
If you prefer manual control:

```bash
# Download release archive
wget https://github.com/ghuntley/cursed/releases/download/v1.0.0-rc1/cursed-v1.0.0-rc1-$(uname -s)-$(uname -m).tar.gz

# Extract and install
tar -xzf cursed-v1.0.0-rc1-*.tar.gz
sudo mv cursed-interpreter /usr/local/bin/
sudo chmod +x /usr/local/bin/cursed-interpreter

# Verify installation
cursed-interpreter --version
```

## 🖥️ Platform-Specific Instructions

### **Linux (Recommended)**

#### **Ubuntu/Debian**
```bash
# Update package list
sudo apt update

# Install dependencies (optional, for building from source)
sudo apt install -y build-essential curl wget

# Install CURSED
curl -sSf https://install.cursedlang.org/v1.0.0-rc1.sh | sh

# Add to PATH (add to ~/.bashrc or ~/.zshrc)
export PATH="$HOME/.cursed/bin:$PATH"
```

#### **CentOS/RHEL/Fedora**
```bash
# Install dependencies  
sudo dnf install -y curl wget gcc

# Install CURSED
curl -sSf https://install.cursedlang.org/v1.0.0-rc1.sh | sh

# Add to PATH
echo 'export PATH="$HOME/.cursed/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

#### **Arch Linux**
```bash
# Dependencies available in base system
# Install CURSED
curl -sSf https://install.cursedlang.org/v1.0.0-rc1.sh | sh

# Add to PATH
echo 'export PATH="$HOME/.cursed/bin:$PATH"' >> ~/.zshrc
```

### **macOS**

#### **Intel Macs (Recommended)**
```bash
# Install via Homebrew (when available)
brew tap cursed-lang/cursed
brew install cursed-lang/cursed/cursed@1.0.0-rc1

# Or manual installation
curl -sSf https://install.cursedlang.org/v1.0.0-rc1.sh | sh
```

#### **Apple Silicon (M1/M2) - Experimental**
```bash
# Note: ARM64 support is experimental in RC1
# Use Intel binary via Rosetta for best compatibility
arch -x86_64 bash -c "$(curl -sSf https://install.cursedlang.org/v1.0.0-rc1.sh)"

# Or native ARM64 (may have issues)
curl -sSf https://install.cursedlang.org/v1.0.0-rc1-arm64.sh | sh
```

### **Windows**

#### **Windows 10/11 (PowerShell)**
```powershell

Invoke-WebRequest -Uri https://install.cursedlang.org/v1.0.0-rc1.ps1 -OutFile install.ps1
```

#### **Windows Subsystem for Linux (WSL)**

## 🔨 Building from Source

### **Prerequisites**
- **Zig 0.15.1 or later** (required)
- **Git** (for cloning repository)
- **5GB disk space** (for build dependencies)

### **Build Process**
```bash
# Clone repository
git clone https://github.com/ghuntley/cursed.git
cd cursed
git checkout v1.0.0-rc1

# Build minimal interpreter (most reliable)
zig build cursed-minimal

# Test the build
./zig-out/bin/cursed-minimal --version
echo 'vibez.spill("Build successful!")' | ./zig-out/bin/cursed-minimal

# Install system-wide (optional)
sudo cp ./zig-out/bin/cursed-minimal /usr/local/bin/cursed-interpreter
```

### **Alternative Build Targets**
```bash
# For development/testing (includes additional tools)
zig build

# For release optimization (if compilation issues are resolved)
zig build -Doptimize=ReleaseFast

# For cross-compilation (experimental)
zig build -Dtarget=x86_64-linux
zig build -Dtarget=aarch64-macos
```

## 🎯 Development Setup

### **Editor Integration**

#### **VS Code (Recommended)**
```bash
# Install VS Code extension
code --install-extension cursed-lang.cursed-vscode-rc1

# Or manually download from marketplace
# Search for "CURSED Language Support"
```

#### **Vim/Neovim**
```vim
" Add to .vimrc or init.vim
Plug 'cursed-lang/vim-cursed'

" Basic syntax highlighting and indentation
filetype plugin indent on
syntax on
```

#### **Emacs**
```elisp
;; Add to .emacs or init.el
(add-to-list 'load-path "~/.emacs.d/cursed-mode")
(require 'cursed-mode)
(add-to-list 'auto-mode-alist '("\\.csd\\'" . cursed-mode))
```

### **Basic Project Structure**
```bash
mkdir my-cursed-project
cd my-cursed-project

# Create main file
cat << 'EOF' > main.csd
// main.csd - My CURSED Project
yeet "vibez"
yeet "mathz"

slay main() tea {
    vibez.spill("Welcome to CURSED v1.0.0-rc1!")
    
    sus result drip = mathz.pow(2, 8)
    vibez.spill("2^8 =", result)
    
    damn "Program completed successfully!"
}
EOF

# Run your project
cursed-interpreter main.csd
```

### **Testing Setup**
```bash
# Create test file
cat << 'EOF' > test.csd
yeet "testz"
yeet "mathz"

test_start("Math Operations")

// Test basic arithmetic
assert_eq_int(2 + 2, 4)
assert_eq_int(mathz.abs(-5), 5)
assert_eq_real(mathz.sqrt(16.0), 4.0)

// Test string operations
yeet "stringz"
assert_eq_str(stringz.concat("Hello", " World"), "Hello World")

print_test_summary()
EOF

# Run tests
cursed-interpreter test.csd
```

## 🔧 Troubleshooting

### **Common Installation Issues**

#### **"Command not found: cursed-interpreter"**
```bash
# Check if installed
ls ~/.cursed/bin/cursed-interpreter

# Add to PATH manually
export PATH="$HOME/.cursed/bin:$PATH"

# Make permanent (add to shell profile)
echo 'export PATH="$HOME/.cursed/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

#### **"Permission denied"**
```bash
# Fix permissions
chmod +x ~/.cursed/bin/cursed-interpreter

# Or if installed system-wide
sudo chmod +x /usr/local/bin/cursed-interpreter
```

#### **"Library not found" (Linux)**
```bash
# Install system dependencies
sudo apt install -y libc6-dev libgcc-s1

# Or on CentOS/RHEL
sudo dnf install -y glibc-devel libgcc
```

### **Build from Source Issues**

#### **"Zig version incompatible"**
```bash
# Check Zig version
zig version
# Need: 0.15.1 or later

# Update Zig
# Visit: https://ziglang.org/download/
```

#### **"LLVM not found" (if building full compiler)**
```bash
# Ubuntu/Debian
sudo apt install -y llvm-16-dev libclang-16-dev

# macOS  
brew install llvm

# Note: For RC1, use minimal build to avoid LLVM issues
zig build cursed-minimal
```

#### **Build hangs or fails**
```bash
# Clean build cache
rm -rf zig-cache/ zig-out/

# Try minimal build
zig build cursed-minimal

# If still fails, report issue with:
zig version
uname -a
# And the error output
```

### **Runtime Issues**

#### **"Module not found" errors**
```bash
# Verify standard library modules are available
cursed-interpreter -c 'yeet "vibez"; vibez.spill("OK")'

# If this fails, reinstall or rebuild
```

#### **Segmentation faults**
```bash
# Run with debugging
gdb cursed-interpreter
(gdb) run your_program.csd
(gdb) bt

# Or run with Valgrind for detailed error
valgrind cursed-interpreter your_program.csd
```

#### **Performance issues**
```bash
# Check if running in debug mode
cursed-interpreter --help | grep -i debug

# For better performance, ensure release build
# (Note: compilation mode may have issues in RC1)
```
