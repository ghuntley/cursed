why cursed hits different

💅

gen z native

keywords that actually make sense in 2024. slay functions, sus variables, lowkey conditionals.

⚡

performance queen

compiles to LLVM IR for that optimized performance. your code runs fast enough to keep up with your thoughts.

🔄

self-hosting era

bootstrap compiler written in rust, but the final compiler is written in cursed itself. that's called eating your own cooking!

🧠

go-like vibes

familiar syntax structure with garbage collection, goroutines (called stan), and channels (dms).

🎯

type safety bestie

static typing with inference. no runtime surprises, just compile-time reality checks.

📦

package ecosystem

import packages with yeet and organize code with the vibe system. modularity that serves.
