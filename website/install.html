quick install

one-liner installation (recommended)

the fastest way to get cursed on your machine. this script will:

  • ✨ download the latest cursed compiler
  • 🎯 install to the right location for your system
  • 📦 set up package manager integration
  • ⚡ configure your path automatically
terminal 
# install cursed with one command
curl -sSL https://install.cursed.dev | sh

# verify installation
cursed --version

# create your first program
echo 'vibe main; slay main() { println("periodt bestie!") }' > hello.csd

# run it!
cursed run hello.csd

manual installation

want more control? install cursed manually for maximum customization

📦 from source

build the latest version directly from the repository

# clone the repo
git clone https://github.com/ghuntley/cursed.git
cd cursed

# build with cargo
cargo build --release

# install to your path
cargo install --path .

# verify it works
cursed --version

🍺 homebrew (macOS)

use homebrew for easy updates and dependency management

# add the cursed tap
brew tap ghuntley/cursed

# install cursed
brew install cursed

# update later with
brew upgrade cursed

🐧 package managers

use your system's package manager for seamless integration

# arch linux (aur)

sudo apt install cursed-lang

sudo dnf install cursed-lang

# build your own image
docker build -t my-cursed-app .

should output: hello bestie! ✨

3

check package manager

should show package management commands and options

4

test compilation

cursed build --emit llvm-ir -o test.ll <(echo 'vibe main; slay main() { }')

should create test.ll with LLVM IR output

troubleshooting common issues

command not found: cursed

your PATH might not be configured. try:

  • restart your terminal
  • run source ~/.bashrc or source ~/.zshrc
  • manually add to PATH: export PATH="$HOME/.cursed/bin:$PATH"

permission denied

installation might need elevated permissions:

  • make sure you have write access to the install directory
  • try using sudo for system-wide installation
  • consider user-local installation instead

llvm not found

cursed needs LLVM for compilation:

  • install LLVM 15+ on your system
  • set LLVM_SYS_150_PREFIX if needed
  • use the bundled LLVM in the installer

slow compilation

first-time compilation might be slow:

  • subsequent builds use incremental compilation
  • use --jobs N for parallel compilation
  • enable optimization caching with --cache-dir
