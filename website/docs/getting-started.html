hello world

let's start with the classic hello world program that serves absolute icon energy:

fr fr this is your first cursed program bestie!
vibe main

yeet "stdlib::io"

slay main() {
    io.println("hello world! it's giving main character energy ✨")
}

save this as hello.csd and run it:

cursed run hello.csd

💡 what's happening here?

  • vibe main - declares this as the main package
  • yeet "stdlib::io" - imports the I/O library (because we yeet imports)
  • slay main() - defines the main function (functions slay in cursed)
  • io.println() - prints text with a newline

package declaration

every cursed file starts with a package declaration:

vibe main        fr fr main package
vibe utils       fr fr custom package
vibe webserver   fr fr another package

imports

import other packages with yeet:

fr fr single import
yeet "stdlib::io"

fr fr multiple imports
yeet (
    "stdlib::io"
    "stdlib::strings"
    fmt "stdlib::format"  fr fr with alias
)

functions

functions in cursed are defined with slay because functions absolutely slay:

basic functions

fr fr simple function
slay greet() {
    io.println("hey bestie! ✨")
}

fr fr function with parameters
slay greet_person(name tea) {
    io.printf("hey {}! you're iconic! ✨\n", name)
}

fr fr function with return value
slay add(x, y normie) normie {
    yolo x + y
}

multiple return values

fr fr function returning multiple values
slay divide(x, y decimal) (decimal, vibes) {
    lowkey y == 0.0 {
        yolo 0.0, sus  fr fr can't divide by zero bestie
    }
    yolo x / y, based
}

fr fr calling function with multiple returns
sus result, ok = divide(10.0, 3.0)
lowkey ok {
    io.printf("result: {}\n", result)
} highkey {
    io.println("division failed bestie")
}

🎯 function naming conventions

  • use snake_case for function names
  • start with lowercase for private functions
  • start with uppercase for public/exported functions
  • make names descriptive and iconic

control flow

control your program's flow with cursed's expressive keywords:

conditional statements

fr fr basic if statement
sus age = 22

lowkey age >= 18 {
    io.println("you're an adult bestie! ✨")
} highkey age >= 13 {
    io.println("teen energy! 😎")
} highkey {
    io.println("smol bean energy 🥺")
}

switch statements

fr fr switch on values
sus mood = "happy"

vibe_check mood {
    mood "happy", "excited":
        io.println("bestie is thriving! ✨")
    mood "sad":
        io.println("sending virtual hugs 🫂")
    mood "stressed":
        io.println("we've all been there bestie")
    basic:
        io.println("neutral energy, and that's ok")
}

loops

fr fr for loop
bestie i := 0; i < 5; i++ {
    io.printf("counting: {}\n", i)
}

fr fr while loop
sus x = 10
periodt x > 0 {
    io.printf("countdown: {}\n", x)
    x--
}

fr fr range loop
sus items = []tea{"apple", "banana", "cherry"}
bestie index, item := flex items {
    io.printf("{}: {}\n", index, item)
}

loop control

fr fr break and continue
bestie i := 0; i < 10; i++ {
    lowkey i == 3 {
        simp  fr fr continue (skip this iteration)
    }
    lowkey i == 8 {
        ghosted  fr fr break (exit loop)
    }
    io.printf("number: {}\n", i)
}
