# CURSED Examples Library

A comprehensive collection of examples demonstrating CURSED programming language features and patterns.

## 🎯 Quick Start Examples

### Hello World
```cursed
vibez.spill("Hello, CURSED!")
```

### Variables and Types
```cursed
sus name tea = "Alice"
sus age normie = 30
sus height drip = 5.8
sus active lit = based

vibez.spill("Name: " + name)
vibez.spill("Age: " + age.(tea))
vibez.spill("Height: " + height.(tea))
vibez.spill("Active: " + active.(tea))
```

### Functions
```cursed
slay greet(name tea) tea {
    damn "Hello, " + name + "!"
}

sus message tea = greet("World")
vibez.spill(message)
```

## 🔍 How to Use Examples

### Running Examples
```bash
# Run example directly
cargo run --bin cursed examples/basic/hello_world.csd

# Compile and run
cargo run --bin cursed -- compile examples/basic/hello_world.csd
./hello_world
```

### Modifying Examples
1. Copy the example to your workspace
2. Modify the code as needed
3. Run with the CURSED compiler
4. Experiment with different approaches
