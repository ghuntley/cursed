# MATHZ Module - Mathematical Operations

Essential mathematical functions for CURSED programs.

## Functions

### Basic Arithmetic
- `abs_normie(x drip) drip` - Absolute value
- `max_normie(a drip, b drip) drip` - Maximum of two numbers
- `min_normie(a drip, b drip) drip` - Minimum of two numbers
- `add_two(a drip, b drip) drip` - Addition
- `subtract_two(a drip, b drip) drip` - Subtraction
- `multiply_two(a drip, b drip) drip` - Multiplication
- `divide_two(a drip, b drip) drip` - Division (returns 0 if dividing by 0)

### Advanced Functions
- `power_int(base drip, exponent drip) drip` - Integer exponentiation
- `factorial(n drip) drip` - Factorial function
- `gcd(a drip, b drip) drip` - Greatest common divisor
- `lcm(a drip, b drip) drip` - Least common multiple

### Utility Functions
- `is_even(n drip) lit` - Check if number is even
- `is_odd(n drip) lit` - Check if number is odd
- `clamp(value drip, min_val drip, max_val drip) drip` - Clamp value to range
- `sign(x drip) drip` - Sign function (-1, 0, or 1)

### Sequences
- `sum_range(start drip, end drip) drip` - Sum of range of numbers
- `fibonacci(n drip) drip` - Nth Fibonacci number

## Usage

```cursed
yeet "mathz"

sus value drip = abs_normie(-42)
sus maximum drip = max_normie(10, 20)
sus fact drip = factorial(5)
sus is_even_check lit = is_even(4)

vibez.spill("Absolute value:", value)
vibez.spill("Maximum:", maximum)
vibez.spill("Factorial:", fact)
vibez.spill("Is even:", is_even_check)
```
