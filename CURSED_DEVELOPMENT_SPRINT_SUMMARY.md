### **Example CURSED Program**
```cursed
sus name tea = "CURSED Programmer"
sus skills []tea = ["coding", "debugging", "slaying"]

slay greet(person tea) tea {
    damn "Hey " + person + ", you're absolutely slaying it! 💯"
}

vibez.spill(greet(name))

bestie skill := flex skills {
    vibez.spill("Skill:", skill)
}
```
