Named Return Values

When you want your return values to have their own moment:

First-Class Functions

Functions are giving first-class citizen vibes in CURSED:

fr fr Higher-order function that returns a function

    mood "+":
        yolo slay(a normie, b normie) normie { yolo a + b }

Anonymous Functions & Closures

When you need a function but don't want to give it the full name recognition:

    fr fr Anonymous function assigned to a variable

    fr fr Immediately invoked function expression (IIFE)
    slay() {
        vibez.spill("This runs immediately and then it's over")

    vibez.spill(incrementCounter())  fr fr 2
