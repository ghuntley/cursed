Getting Started Beginner First Steps

Let's get this bread with your first CURSED program! It's giving hello world energy but make it GenZ.

Your First Program

hello.csd
vibe main

yeet "vibez"

slay main() {
    vibez.spill("OMG hiiii bestie!")
}

How to run your code

  1. Save the code above to a file called hello.csd
  2. Open a terminal and navigate to your file's location
  3. Run the command: cursed run hello.csd
  4. See your code slay with the output: OMG hiiii bestie!

What's Going On?

Let's break down this program line by line:

  • �� vibe main - Declares that this file belongs to the main package
  • 🚀 yeet "vibez" - Imports the vibez package (like fmt in Go)
  • �� slay main() { ... } - Defines the main function, which runs automatically
  • �� vibez.spill("OMG hiiii bestie!") - Prints text to the console

CURSED vs Traditional Languages

Here's what the same program would look like in some other languages:
