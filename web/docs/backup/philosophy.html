Design Philosophy

CURSED isn't just a vibe check - it's a whole programming language aesthetic. Let's spill the tea on our core design principles that make CURSED absolutely slay.

Core Principles

  • Syntactic Familiarity with a Twist: We took Go's grammar (which was already giving clean girl aesthetic) and made it pop off with GenZ slang keywords that are living rent-free in our heads.
  • Self-Hosting: The compiler is designed to compile itself eventually - we're manifesting that girlboss energy through bootstrapping.
  • Pragmatic Absurdity: It's giving esoteric language vibes but make it actually useful? Like yes, we're quirky, but we still get things done.
  • Compilation Efficiency: The code might be chaotic, but the compiled result doesn't flop. We want efficiency that slays AND readability that stays.
