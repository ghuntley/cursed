Toolchain

Let's get this bread with the CURSED toolchain! These developer tools are bussin' and will help you slay your coding projects.

Core CLI Commands

Command Description Example
cursed build Compile packages and dependencies cursed build main.csd
cursed run Compile and run cursed run app.csd
cursed test Run tests cursed test ./...
cursed fmt Format code (like gofmt) cursed fmt ./...
cursed get Download and install packages cursed get github.com/user/package
cursed doc Show documentation cursed doc vibez
cursed vet Report likely mistakes cursed vet ./...
Building Projects
fr fr Compile a single file
$ cursed build main.csd

fr fr Compile and link a package and dependencies
$ cursed build

fr fr Cross compile for different OS/architecture
$ CS_GOOS=windows CS_GOARCH=amd64 cursed build

fr fr Build with race detector
$ cursed build -race

fr fr Build with optimization levels
$ cursed build -opt=2

Project Structure

my-cursed-project/
├── .cursed/           fr fr Project metadata
├── cmd/               fr fr Command line applications
│   └── server/
│       └── main.csd   fr fr Entry point for server application
├── internal/          fr fr Private application code
│   ├── auth/
│   └── db/
├── pkg/               fr fr Public library code
│   └── models/
├── test/              fr fr Additional test files
├── web/               fr fr Web assets
├── views/             fr fr Templates
├── go.mod             fr fr Go modules compatibility
└── README.md          fr fr Project documentation
Package Management
fr fr Initialize a module
$ cursed mod init github.com/username/project

fr fr Add a dependency
$ cursed get github.com/username/somelib

fr fr Update dependencies
$ cursed get -u ./...

fr fr Clean up unused dependencies
$ cursed mod tidy

IDE Support

CURSED has integrations with popular editors that are giving productivity vibes:

  • VS Code: Install the "CURSED Language" extension
  • JetBrains: CURSED plugin available in the marketplace
  • Vim/Neovim: CURSED syntax and tooling plugins
  • Emacs: cursed-mode for syntax highlighting and linting

Debugging Tools

fr fr Run with verbose logging
$ cursed run -v main.csd

fr fr Debug mode with extended info
$ cursed run -debug main.csd

fr fr Profiling
$ cursed test -cpuprofile=cpu.prof -memprofile=mem.prof

fr fr Analyze profiles
$ cursed tool pprof cpu.prof

fr fr Trace execution
$ cursed trace main.csd

Common Build Flags

Flag Purpose
-o output Name output file
-race Enable race detection
-tags Build with specific tags
-ldflags Pass arguments to linker
-gcflags Pass arguments to compiler
