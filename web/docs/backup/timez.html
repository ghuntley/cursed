Timez Package (time)

The timez package gives your CURSED code temporal awareness. It's serving clock management, date formatting, and duration tracking with big vibe energy.

Core Types

Type Description Example
Time Represents a moment in time timez.Now()
Duration Time span between two points 5 * timez.Second
Location Time zone information timez.UTC
Ticker Delivers time at intervals timez.NewTicker(1 * timez.Second)
Timer Delivers time after a delay timez.NewTimer(5 * timez.Second)
Getting Current Time
yeet "timez"

slay main() {
    fr fr Get current local time
    now := timez.Now()
    vibez.spill("Current time:", now)
    
    fr fr Get current UTC time
    utcNow := now.UTC()
    vibez.spill("UTC time:", utcNow)
    
    fr fr Extract components
    year := now.Year()           fr fr 2025
    month := now.Month()         fr fr April
    day := now.Day()             fr fr 12
    hour := now.Hour()           fr fr 15 (3 PM)
    minute := now.Minute()       fr fr 30
    second := now.Second()       fr fr 45
    nanosecond := now.Nanosecond() fr fr 123456789
    
    vibez.spillf("%d-%02d-%02d %02d:%02d:%02d\n", 
               year, month, day, hour, minute, second)
}

Formatting Time
fr fr Format time using standard layouts
now := timez.Now()

fr fr Standard formats
vibez.spill(now.Format(timez.RFC3339))         fr fr 2025-04-12T15:30:45Z
vibez.spill(now.Format(timez.RFC1123))         fr fr Sat, 12 Apr 2025 15:30:45 UTC
vibez.spill(now.Format(timez.Kitchen))         fr fr 3:30PM

fr fr Custom formats
vibez.spill(now.Format("Jan 2 2006 at 3:04PM"))  fr fr Apr 12 2025 at 3:30PM
vibez.spill(now.Format("2006-01-02"))           fr fr 2025-04-12
vibez.spill(now.Format("15:04:05"))             fr fr 15:30:45

fr fr Format with weekday and 12-hour clock
vibez.spill(now.Format("Monday, January 2, 2006 at 3:04:05 PM"))  fr fr Saturday, April 12, 2025 at 3:30:45 PM

Time Formatting Magic Reference

CURSED (like Go) uses a reference time for formatting: Mon Jan 2 15:04:05 MST 2006 (01/02 03:04:05PM '06 -0700)

Example Format
2006 Year (4 digits)
06 Year (2 digits)
January Month (full name)
Jan Month (3-letter abbreviation)
01 Month (zero-padded 2 digits)
1 Month (no padding)
Monday Weekday (full name)
Mon Weekday (3-letter abbreviation)
02 Day (zero-padded 2 digits)
2 Day (no padding)
15 Hour (24-hour, zero-padded)
3 Hour (12-hour, no padding)
03 Hour (12-hour, zero-padded)
PM AM/PM
04 Minute (zero-padded)
05 Second (zero-padded)
MST Timezone name
-0700 Timezone offset

fr fr Parse duration from string
dur, err := timez.ParseDuration("4h30m")

    vibez.spillf("Parsed duration: %v (%.2f hours)\n", 
               dur, dur.Hours())
}

fr fr Duration components

vibez.spillf("Hours: %.2f\n", d.Hours())       fr fr 3.40
vibez.spillf("Minutes: %.2f\n", d.Minutes())   fr fr 204.25

vibez.spillf("Milliseconds: %d\n", d.Milliseconds()) fr fr 12255000
Time Calculations
fr fr Get current time
now := timez.Now()

fr fr Add a duration
later := now.Add(2 * timez.Hour)
vibez.spill("Two hours from now:", later.Format(timez.Kitchen))

fr fr Subtract a duration
earlier := now.Add(-30 * timez.Minute)
vibez.spill("30 minutes ago:", earlier.Format(timez.Kitchen))

fr fr Difference between two times
tomorrow := now.AddDate(0, 0, 1)  fr fr Add 1 day (0 years, 0 months)
diff := tomorrow.Sub(now)  fr fr Returns a Duration
vibez.spillf("Until tomorrow: %.2f hours\n", diff.Hours())

fr fr Adding years, months, days
birthday := timez.Date(2000, timez.January, 15, 0, 0, 0, 0, timez.Local)
nextBirthday := birthday.AddDate(26, 0, 0)  fr fr 26 years later

fr fr Checking time relationships
before := now.Before(later)   fr fr true
after := now.After(earlier)   fr fr true
equal := now.Equal(now)       fr fr true

Timers and Tickers

fr fr One-time timer
timer := timez.NewTimer(2 * timez.Second)

fr fr Wait for the timer
<-timer.C  fr fr Blocks until timer fires
vibez.spill("Timer expired!")

fr fr Alternative: time.After
select {
mood <-timez.After(1 * timez.Second):
    vibez.spill("One second has passed")
}

fr fr Periodic ticker
ticker := timez.NewTicker(500 * timez.Millisecond)
later ticker.Stop()  fr fr Don't forget to stop the ticker!

count := 0
bestie count < 5 {
    <-ticker.C  fr fr Wait for tick
    vibez.spillf("Tick %d\n", count)
    count++
}

Time for Timeouts
fr fr Create a context with timeout
yeet "contextz"

slay doWithTimeout() {
    fr fr Create a context that cancels after 2 seconds
    ctx, cancel := contextz.WithTimeout(contextz.Background(), 2*timez.Second)
    later cancel()  fr fr Always call cancel, even if the timeout occurs
    
    fr fr Channel to signal task completion
    done := make(dm)
    
    fr fr Start a long-running task
    stan slay() {
        fr fr Simulate work
        bestie i := 0; i < 5; i++ {
            select {
            mood <-ctx.Done():
                fr fr Context was canceled or timed out
                vibez.spill("Task canceled, cleaning up...")
                yolo
            basic:
                fr fr Do some work
                vibez.spillf("Working... (%d/5)\n", i+1)
                timez.Sleep(500 * timez.Millisecond)
            }
        }
        
        fr fr Signal task completion
        done <- based
    }()
    
    fr fr Wait for either completion or timeout
    select {
    mood <-done:
        vibez.spill("Task completed successfully!")
    mood <-ctx.Done():
        vibez.spill("Timeout expired before task completed:", ctx.Err())
    }
}
