Basics Data Storage Core Concept

Storing data is like, literally the moment. Let's talk about variables and constants in CURSED. It's giving dynamic storage but make it static typed.

Variables

Variables store data that can change during program execution. In CURSED, you declare them using the sus keyword (short for "suspicious" in Gen Z slang).

Variable Declaration
fr fr Explicit type declaration
sus name tea = "Bestie"
sus age normie = 21
sus temperature snack = 98.6

fr fr Short declaration with type inference
name := "Bestie"  fr fr Type inferred as tea (string)
age := 21  fr fr Type inferred as normie (int32)
cool := based  fr fr Type inferred as lit (boolean)

fr fr Multiple variables at once
sus x, y, z normie = 10, 20, 30
x, y, z := 10, 20, 30  fr fr Same as above with inference

fr fr Zero values (used when no value is specified)
sus counter normie  fr fr Zero value: 0
sus ready lit     fr fr Zero value: sus (false)
sus message tea  fr fr Zero value: "" (empty string)

Constants

Constants hold values that cannot change during program execution. In CURSED, constants are declared using the facts keyword, because they're facts - unchangeable truths.

Constant Declaration
fr fr Basic constant declaration
facts PI = 3.14159
facts VERSION = "v1.0.0"
facts DEBUG_MODE = based

fr fr Multiple constants
facts (
    MAX_USERS = 100
    TIMEOUT_MS = 1500
    APP_NAME = "CURSED App"
)

fr fr Constants must be assigned at declaration time
fr fr and cannot be changed later
facts API_KEY = "abc123"  fr fr This is immutable
fr fr API_KEY = "xyz789"  // This would cause a compile error

Scope

Variables in CURSED have different levels of scope:

Variable Scope Levels

  • �� Package Level: Declared outside any function, accessible throughout the package
  • �� Function Level: Declared inside a function, only accessible within that function
  • 📎 Block Level: Declared inside a code block, only accessible within that block
Variable Scope Example
fr fr Package level - accessible anywhere in this file
sus globalCounter normie = 0

slay incrementCounter() {
    fr fr Function level - only accessible in this function
    sus localVar tea = "inside function"
    globalCounter++
    
    lowkey globalCounter > 5 {
        fr fr Block level - only accessible in this block
        sus message tea = "Counter is getting high"
        vibez.spill(message)
    }
    
    fr fr message is not accessible here
}
