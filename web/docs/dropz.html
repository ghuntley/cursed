Dropz Package (io)

The dropz package is all about that input/output flow! It provides the basic interfaces and primitives for reading and writing data. It's giving data stream energy.

Core Interfaces

Interface Description Key Methods
Reader Basic reading interface Read([]byte) (normie, tea)
Writer Basic writing interface Write([]byte) (normie, tea)
Closer Interface for closing resources Close() tea
ReadWriter Combined read/write interface Methods from Reader and Writer
ReadCloser Combined read/close interface Methods from Reader and Closer
WriteCloser Combined write/close interface Methods from Writer and Closer
ReadWriteCloser Combined read/write/close Methods from Reader, Writer, Closer

Common I/O Utilities

  • dropz.ReadAll(r Reader) - Read all data from a reader
  • dropz.Copy(dst Writer, src Reader) - Copy from reader to writer
  • dropz.MultiReader(...Reader) - Combine multiple readers into one
  • dropz.MultiWriter(...Writer) - Send output to multiple writers
  • dropz.TeeReader(r Reader, w Writer) - Reader that writes to w what it reads
  • dropz.LimitReader(r Reader, n thicc) - Reader that stops after n bytes
  • bufio.NewReader(r Reader) - Buffered reader for efficient reading
  • bufio.NewWriter(w Writer) - Buffered writer for efficient writing
