Language Overview

CURSED is an esoteric programming language designed to combine the simplicity and efficiency of Go with the contemporary linguistic flair of Generation Z slang. It's statically typed, garbage collected, and features built-in concurrency support.

Core Design Principles

  • Syntactic Familiarity with a Twist: Go-like grammar with Gen Z slang keywords
  • Self-Hosting: Designed to compile itself through bootstrapping
  • Pragmatic Absurdity: Esoteric by design yet fully functional
  • Compilation Efficiency: Produces efficient code without sacrificing readability

Language Characteristics

  • ✨ Static typing with type inference
  • 🗑️ Automatic garbage collection
  • ⚡ Built-in concurrency primitives
  • 📦 Package-based code organization
  • 💬 Expression-based constructs

Goroutines and Channels
vibe main

yeet "vibez"
yeet "timez"

slay say(msg tea, ch dm) {
    timez.sleep(100 * timez.millisecond)
    ch <- msg  fr fr Send message to channel
}

slay main() {
    messages := dm(2)  fr fr Create buffered channel
    
    stan say("what's good", messages)  fr fr Run in goroutine
    stan say("it's giving concurrency", messages)
    
    vibez.spill(<-messages)  fr fr Receive from channel
    vibez.spill(<-messages)
}
Generics Example
vibe main

yeet "vibez"

slay mapSlice<T, U>(items []T, fn slay(T) U) []U {
    result := make([]U, len(items))
    bestie i := 0; i < len(items); i++ {
        result[i] = fn(items[i])
    }
    yolo result
}

slay main() {
    nums := []normie{1, 2, 3, 4, 5}
    squared := mapSlice(nums, slay(x normie) normie {
        yolo x * x
    })
    vibez.spill(squared)  fr fr Output: [1 4 9 16 25]
}
