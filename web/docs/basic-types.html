Type Conversions That Pass the Vibe Check

CURSED requires explicit type conversions to keep things safe and clear:

sus intValue normie = 42
sus floatValue snack = snack(intValue)  fr fr int to float conversion
sus biggerInt thicc = thicc(intValue)  fr fr int32 to int64 conversion
sus stringValue tea = tea(intValue)  fr fr int to string conversion
sus charValue sip = sip(65)  fr fr int to char conversion (becomes 'A')

Zero Values

When variables are declared without initialization, they get these default zero values:

lit (boolean) sus (false)
Numeric types 0
