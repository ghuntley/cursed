Getting Started

Welcome to CURSED! This guide will help you install the CURSED compiler, set up your environment, and write your first program.

Installation

You can install CURSED using the following commands:

$ git clone https://github.com/ghuntley/cursed.git
$ cd cursed
$ make install

Your First CURSED Program

Create a file named hello.csd with the following content:

vibe main

slay main() {
    print("no cap, hello world!")
}

Run your program with:

$ cursed run hello.csd

You should see the following output:

no cap, hello world!

Language Basics

CURSED is a compiled, statically typed language with a syntax inspired by Go but using Gen Z slang as keywords.

Keywords

Here's a mapping between traditional programming keywords and CURSED keywords:

Traditional CURSED Description
package vibe Declares the package name
import yeet Imports other packages
func slay Defines a function
return yolo Returns a value from a function
var sus Declares a variable
const facts Declares a constant
if lowkey Conditional statement
else highkey Alternative branch
for bestie Loop statement
while periodt While loop
switch vibe_check Switch statement
case mood Case in a switch statement
default basic Default case
break ghosted Breaks out of a loop
continue simp Continues to next iteration
type be_like Defines a type
struct squad Composite data type
interface collab Interface type
map tea Map type
chan dm Channel type
go stan Starts a goroutine
range flex Iterates over elements
defer later Defers execution
true based Boolean true value
false sus Boolean false value
nil cap Zero value for pointers, etc.

Types

CURSED has several built-in types:

Type Description Example
lit Boolean based, sus
smol 8-bit integer 127, -128
mid 16-bit integer 32767, -32768
normie 32-bit integer 2147483647, -2147483648
thicc 64-bit integer 9223372036854775807, -9223372036854775808
snack 32-bit float 3.14, -2.5
meal 64-bit float 3.141592653589793, -2.5
tea String "hello", "world"
sip Character 'a', '世', '\n'

Variables

You can declare variables using the sus keyword or with short declaration syntax:

sus name tea = "CURSED"
sus age normie = 25

fr fr Short declaration syntax
vibe := "check"    fr fr Type inferred as tea
count := 42       fr fr Type inferred as normie

Functions

Functions are declared using the slay keyword:

slay add(x, y normie) normie {
    yolo x + y
}

slay greet(name tea) {
    print("Hey, " + name + "!")
}

slay divide(x, y snack) (snack, tea) {
    lowkey y == 0 {
        yolo 0, "can't divide by zero"
    }
    yolo x / y, ""
}
