# 🎉 CURSED v1.0.0-rc2 Community Announcement

## The Gen Z Programming Language is Ready for Testing! 

**CURSED v1.0.0-rc2** is now available for community testing and feedback. This release candidate represents **92% completion** toward the stable v1.0 release and includes enterprise-grade reliability, memory safety, and performance.

## 🚀 What Makes CURSED Special?

### **Unique Gen Z Slang Syntax**
```cursed
sus name tea = "Developer"        // Variables with 'sus'  
slay greet(person tea) tea {      // Functions with 'slay'
    damn "Hey " + person + "!"    // Return with 'damn'
}
bestie i < 10 { i = i + 1 }      // Loops with 'bestie'
vibez.spill("Hello World!")       // Output with 'vibez.spill'
```

### **Serious Performance & Safety**
- ⚡ **Lightning Fast**: Sub-second builds (0.05-0.2s)
- 🛡️ **Memory Safe**: Zero memory leaks (Valgrind confirmed)
- 🧠 **Advanced Types**: Generics, constraints, interfaces
- 🔄 **Built-in Concurrency**: Goroutines and channels
- 🌍 **Cross-Platform**: Linux, macOS, Windows, WebAssembly

## 🧪 Try CURSED Today

### **Quick Installation**
```bash
git clone https://github.com/ghuntley/cursed.git
cd cursed

```

### **Example Programs to Try**
```cursed
// Variables and arithmetic
sus x normie = 10
sus y normie = 20
sus result normie = x + y
vibez.spill("Sum:", result)

// Functions  
slay factorial(n normie) normie {
    ready n <= 1 {
        damn 1
    } otherwise {
        damn n * factorial(n - 1)
    }
}

// Arrays and loops
sus numbers []normie = [1, 2, 3, 4, 5]
sus total normie = 0
sus i normie = 0
bestie i < 5 {
    total = total + numbers[i]
    i = i + 1
}
vibez.spill("Array sum:", total)
```
