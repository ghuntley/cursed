# CURSED Community Events & Engagement Programs

## Signature Events

### CURSED Jam (Quarterly Game Jam)
**Theme:** "Build Something Cursed"  
**Duration:** 48 hours  
**Format:** Global online event

#### Event Structure
- **Friday 6 PM UTC:** Theme announcement and kickoff
- **Saturday-Sunday:** Development period with live support
- **Monday 6 PM UTC:** Submission deadline
- **Following Week:** Community voting and judging

#### Categories
- **Best Overall** - Most impressive complete project
- **Most Creative** - Innovative use of CURSED features
- **Best Newcomer** - Outstanding first-time participant
- **Community Choice** - Highest community votes
- **Technical Excellence** - Best code quality and architecture

#### Resources Provided
- Starter project templates
- Real-time Discord support channels
- Live coding streams by core team
- Resource sharing and collaboration spaces

#### Prizes
- **Grand Prize:** $1000 + CURSED swag + featured on homepage
- **Category Winners:** $500 + recognition + interview opportunity
- **All Participants:** Digital certificate + contributor status

### Monthly CURSED Workshop Series
**Format:** 90-minute interactive sessions  
**Schedule:** Second Saturday of each month

#### Workshop Rotation
1. **"Getting Started with CURSED"** - Beginner fundamentals
2. **"Web Development in CURSED"** - Building REST APIs and web services
3. **"Systems Programming"** - Low-level programming and performance
4. **"Concurrency Patterns"** - Goroutines, channels, and async programming
5. **"CURSED Toolchain Mastery"** - LSP, debugging, and advanced tooling
6. **"Contributing to CURSED"** - Open source contribution workshop

#### Workshop Format
- 20 minutes: Concept introduction and slides
- 50 minutes: Hands-on coding with instructor guidance
- 15 minutes: Q&A and project sharing
- 5 minutes: Wrap-up and next steps

### Weekly Community Sync
**Schedule:** Every Friday 3 PM UTC  
**Duration:** 30 minutes  
**Format:** Voice + screen sharing

#### Agenda Template
1. **Community Updates** (5 min) - Recent achievements and news
2. **Contributor Spotlight** (10 min) - Feature a community member
3. **Technical Discussion** (10 min) - Discuss current developments
4. **Open Floor** (5 min) - Community questions and announcements

## Community Challenges

### Monthly Code Challenge
**Theme-based programming challenges**

#### Challenge Themes
- **Performance Month** - Optimize for speed and memory
- **Creativity Month** - Most innovative use of language features
- **Collaboration Month** - Best team project
- **Newcomer Month** - Designed for first-time participants

#### Challenge Format
- Problem announcement first Monday of month
- Two-week development period
- Community voting and judging
- Winners featured and recognized

### Advent of CURSED
**25-day December coding challenge**

#### Daily Structure
- New programming puzzle each day
- Increasing difficulty throughout month
- Community discussion and solution sharing
- Leaderboard tracking and recognition

#### Support Resources
- Daily hint releases
- Community solution discussions
- Beginner-friendly alternative problems
- Live-streamed solution walkthroughs

## Virtual Conferences

### CURSED Con (Annual)
**Two-day virtual conference celebrating CURSED community**

#### Day 1: Technical Content
- Keynote: "State of CURSED" by core team
- Technical talks by community experts
- Workshop sessions and hands-on learning
- Panel discussions on language future

#### Day 2: Community Focus
- Contributor recognition ceremony
- Community project showcases
- Networking sessions and breakouts
- Planning sessions for next year

#### Content Tracks
- **Language Development** - Compiler and runtime talks
- **Application Development** - Real-world usage stories
- **Community & Education** - Teaching and learning experiences
- **Ecosystem & Tooling** - IDE, libraries, and framework development
