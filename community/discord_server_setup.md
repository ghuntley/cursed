## Community Guidelines

### Core Values 🔥
1. **Respect the Vibe** - Keep discussions constructive and inclusive
2. **Help Others Level Up** - Share knowledge generously  
3. **Stay Curious** - Ask questions, experiment, learn together
4. **No Toxic Energy** - Zero tolerance for harassment or discrimination
5. **Keep It CURSED** - Embrace the playful Gen Z language culture

### Rules
1. **No Spam or Self-Promotion** - Quality over quantity
2. **Use Appropriate Channels** - Keep topics organized
3. **Search Before Asking** - Check pinned messages and #help-desk
4. **No NSFW Content** - Keep it professional and welcoming
5. **Respect Intellectual Property** - Don't share proprietary code
6. **English Primary** - Use English for main channels (language channels for others)

### Moderation Policy
- **Strike System**: Warn → Mute → Temp Ban → Permanent Ban  
- **Appeals Process**: DM moderators for rule clarification
- **Community Reports**: React with 🚨 for moderator attention
