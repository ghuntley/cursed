# CURSED Discord Server Infrastructure

## Server Structure

### General Channels
- **#welcome** - Welcome message and community rules
- **#announcements** - Official CURSED updates and releases
- **#general** - General discussion about CURSED
- **#showcase** - Share projects built with CURSED
- **#resources** - Links to tutorials, docs, and helpful resources

### Development Channels
- **#help-beginners** - Support for new CURSED developers
- **#help-advanced** - Complex technical discussions
- **#feature-requests** - Community feature suggestions
- **#bug-reports** - Bug reporting with template
- **#code-review** - Peer code reviews and feedback

### Contribution Channels
- **#contributors** - Contributors-only channel
- **#good-first-issues** - Issues perfect for newcomers
- **#rfc-discussion** - Discuss language RFC proposals
- **#pull-requests** - PR discussions and updates

### Special Interest
- **#cursed-jam** - Game jam and hackathon coordination
- **#performance** - Performance optimization discussions
- **#stdlib-development** - Standard library discussions
- **#tooling** - IDE plugins and development tools

### Community Events
- **#weekly-sync** - Weekly community check-ins
- **#workshop-announcements** - Educational workshop notifications
- **#mentorship** - Mentor-mentee matching and discussions

## Moderation Structure

### Roles
- **Core Team** - CURSED maintainers and project leaders
- **Moderators** - Community moderation team
- **Contributors** - Active GitHub contributors
- **Mentors** - Experienced developers helping newcomers
- **Community Champions** - Active community members

### Moderation Guidelines
- Enforce Code of Conduct consistently
- 3-strike system for violations
- Immediate ban for harassment or spam
- Log all moderation actions
- Monthly moderator training sessions

## Bot Configuration

### Welcome Bot
- Automated welcome message with getting started guide
- Role assignment based on experience level
- Resource links for new members

### Support Bot
- Automatic issue triage from #bug-reports
- FAQ responses for common questions
- GitHub integration for issue tracking

### Event Bot
- Workshop and event scheduling
- Reminder notifications
- RSVP management for community events
