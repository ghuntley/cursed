# Getting Started with CURSED v1.0.0-rc1

## 🎯 Quick Start (5 Minutes)

Welcome to CURSED! This guide will get you writing and running CURSED code in 5 minutes.

### **Your First CURSED Program**
```cursed
// hello.csd
vibez.spill("Hello, CURSED World! 🚀")
```

**Run it:**
```bash
cursed-interpreter hello.csd
```

**Expected Output:**
```
Hello, CURSED World! 🚀
```

Congratulations! You've just run your first CURSED program! ✨

## 💡 Understanding CURSED Syntax

CURSED uses Gen-Z inspired keywords to make programming more expressive and fun:

### **Variables**
```cursed
sus name tea = "Alice"        // String variable
sus age drip = 25            // Integer variable  
sus height real = 5.6        // Float variable
sus active lit = based       // Boolean variable (based = true, cap = false)

vibez.spill("Name:", name, "Age:", age, "Active:", active)
```

### **Functions**
```cursed
// Basic function
slay greet(name tea) tea {
    damn "Hello, " + name + "!"  // 'damn' is return
}

// Function with multiple parameters
slay add_numbers(a drip, b drip) drip {
    damn a + b
}

// Call functions
sus greeting tea = greet("Developer")
sus sum drip = add_numbers(10, 32)
vibez.spill(greeting)  // Output: Hello, Developer!
vibez.spill("Sum:", sum)  // Output: Sum: 42
```

### **Control Flow**
```cursed
sus score drip = 85

// Conditional statements
ready (score >= 90) {
    vibez.spill("Excellent! A+ grade")
} otherwise ready (score >= 80) {
    vibez.spill("Good job! B grade")  
} otherwise {
    vibez.spill("Keep studying!")
}

// Loops
bestie (sus i drip = 1; i <= 5; i += 1) {
    vibez.spill("Count:", i)
}
```

## 🏗️ Data Structures

### **Arrays**

## 🔀 Concurrency (Basic)

CURSED makes concurrent programming approachable:

### **Goroutines**
```cursed
yeet "concurrenz"

// Simple concurrent function
slay worker(id drip) tea {
    vibez.spill("Worker", id, "starting")
    // Simulate some work
    sus result drip = id * id
    vibez.spill("Worker", id, "result:", result)
    damn "Done"
}

// Launch concurrent goroutines
go worker(1)
go worker(2)  
go worker(3)

// Main thread continues
vibez.spill("All workers started!")
```

### **Channels**
```cursed
yeet "concurrenz"

// Create a channel
sus ch chan<drip> = make_channel()

// Send data in a goroutine
go {
    ch <- 42
    vibez.spill("Sent 42 to channel")
}

// Receive data
sus value drip = <-ch
vibez.spill("Received:", value)  // Output: Received: 42
```

## 🛠️ Error Handling

CURSED has a structured error handling system:

### **Basic Error Handling**
```cursed
// Function that might fail
slay divide(a drip, b drip) yikes<tea> {
    ready (b == 0) {
        yikes "Cannot divide by zero"
    }
    damn a / b
}

// Handle potential errors
sus result drip = divide(10, 2) fam {
    when "Cannot divide by zero" -> {
        vibez.spill("Error: Division by zero!")
        damn 0  // Default value
    }
    when _ -> {
        vibez.spill("Unknown error occurred")
        damn 0
    }
}

vibez.spill("Result:", result)  // Output: Result: 5
```

### **Error Propagation**
```cursed
slay safe_operation() yikes<tea> {
    sus value drip = divide(10, 0)?  // Propagate error with ?
    damn "Operation successful: " + value
}

// Handle at higher level
sus message tea = safe_operation() fam {
    when "Cannot divide by zero" -> damn "Operation failed safely"
}

vibez.spill(message)  // Output: Operation failed safely
```

## 🎯 Real-World Example: Simple Calculator

Let's build a practical example that demonstrates multiple CURSED features:

```cursed
// calculator.csd
yeet "vibez"
yeet "stringz"
yeet "testz"

// Calculator operations
squad Calculator {
    history []tea  // Keep track of operations
}

slay Calculator.new() Calculator {
    damn Calculator{ history: [] }
}

slay Calculator.add(self *Calculator, a real, b real) real {
    sus result real = a + b
    sus operation tea = stringz.format("{} + {} = {}", [a, b, result])
    // Note: Array append needs implementation in stdlib
    vibez.spill("Operation:", operation)
    damn result
}

slay Calculator.subtract(self *Calculator, a real, b real) real {
    sus result real = a - b
    sus operation tea = stringz.format("{} - {} = {}", [a, b, result])
    vibez.spill("Operation:", operation)
    damn result
}

slay Calculator.multiply(self *Calculator, a real, b real) real {
    sus result real = a * b
    sus operation tea = stringz.format("{} × {} = {}", [a, b, result])
    vibez.spill("Operation:", operation)  
    damn result
}

slay Calculator.divide(self *Calculator, a real, b real) yikes<real> {
    ready (b == 0.0) {
        yikes "Division by zero is not allowed"
    }
    sus result real = a / b
    sus operation tea = stringz.format("{} ÷ {} = {}", [a, b, result])
    vibez.spill("Operation:", operation)
    damn result
}

// Main program
slay main() tea {
    vibez.spill("🧮 CURSED Calculator v1.0")
    vibez.spill("==========================")
    
    sus calc Calculator = Calculator.new()
    
    // Perform calculations
    sus sum real = calc.add(10.5, 7.3)
    sus difference real = calc.subtract(20.0, 8.5)
    sus product real = calc.multiply(6.0, 7.0)
    
    // Handle potential division by zero
    sus division_result real = calc.divide(15.0, 3.0) fam {
        when "Division by zero is not allowed" -> {
            vibez.spill_error("❌ Error: Cannot divide by zero!")
            damn 0.0
        }
    }
    
    vibez.spill("\n📊 Final Results:")
    vibez.spill("Sum:", sum)
    vibez.spill("Difference:", difference)
    vibez.spill("Product:", product)
    vibez.spill("Division:", division_result)
    
    damn "Calculator operations completed!"
}

// Run the calculator
main()
```

**Run the calculator:**
```bash
cursed-interpreter calculator.csd
```

## 🔍 Debugging Tips

### **Common Beginner Mistakes**

#### **1. Forgetting to import modules**
```cursed
// ❌ Wrong - will fail
sus result drip = mathz.abs(-5)

// ✅ Correct  
yeet "mathz"
sus result drip = mathz.abs(-5)
```

#### **2. Incorrect variable types**
```cursed
// ❌ Wrong - type mismatch
sus age tea = 25  // tea is for strings

// ✅ Correct
sus age drip = 25  // drip is for integers
```

#### **3. Missing return statement**
```cursed
// ❌ Wrong - no return
slay get_greeting() tea {
    vibez.spill("Creating greeting...")
    // Missing: damn "Hello!"
}

// ✅ Correct
slay get_greeting() tea {
    vibez.spill("Creating greeting...")
    damn "Hello!"
}
```

### **Debug Output**
```cursed
// Use vibez.spill for debugging
slay problematic_function(x drip) drip {
    vibez.spill("DEBUG: x =", x)  // Debug output
    sus result drip = x * 2
    vibez.spill("DEBUG: result =", result)  // Debug output
    damn result
}
```
