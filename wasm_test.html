About CURSED WASM:

This is a WebAssembly compilation of the CURSED programming language compiler. It demonstrates:

  • ✅ Thread-free compilation (no std.Thread dependencies)
  • ✅ Filesystem-free operation (no std.fs dependencies)
  • ✅ Basic CURSED syntax support (vibez.spill, variables, functions)
  • ✅ Tokenization and syntax checking
  • ✅ Memory-safe WASM execution

Supported CURSED Features:

  • vibez.spill("text") - Print text
  • sus variable = value - Declare variables (planned)
  • slay function() { ... } - Function definitions (planned)
  • Basic arithmetic and string operations (planned)
