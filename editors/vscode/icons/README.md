# CURSED VS Code Extension Icons

This directory contains the icon files for the CURSED VS Code extension.

## Required Icons

### Extension Icon
- `cursed-icon.png` (128x128) - Main extension icon for the marketplace

### Theme Icons  
- `cursed-light.svg` - CURSED file icon for light themes
- `cursed-dark.svg` - CURSED file icon for dark themes

## Icon Guidelines

- **Extension Icon**: Should be 128x128 pixels, PNG format
- **File Icons**: Should be SVG format for scalability
- **Design**: Should reflect the "fire" theme of CURSED 🔥
- **Colors**: Use vibrant colors that work well in both light and dark themes

## Placeholder Icons

For now, VS Code will use default ThemeIcon references like:
- `$(fire)` for CURSED-related items
- `$(file-code)` for source files
- `$(package)` for dependencies
- `$(tools)` for build operations

## Future Enhancement

Custom SVG icons can be added here to replace the placeholder theme icons for a more branded experience.
