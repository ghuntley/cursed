# CURSED Language Tutorials

Welcome to the comprehensive CURSED programming language tutorial series! This tutorial series is designed to take you from a complete beginner to an advanced CURSED developer.

## Tutorial Series Overview

### 📚 [Beginner Series](beginner/README.md)
Perfect for those new to CURSED or programming in general.

1. **[Getting Started](beginner/01-getting-started.md)** - Installation, setup, and your first CURSED program
2. **[Basic Syntax](beginner/02-basic-syntax.md)** - Gen Z keywords, comments, and language fundamentals
3. **[Variables and Types](beginner/03-variables-types.md)** - Data types, variable declarations, and type inference
4. **[Functions](beginner/04-functions.md)** - Function definitions, parameters, and return values
5. **[Control Flow](beginner/05-control-flow.md)** - Conditionals, loops, and program flow
6. **[Data Structures](beginner/06-data-structures.md)** - Arrays, slices, structs, and maps
7. **[Error Handling](beginner/07-error-handling.md)** - Basic error handling patterns

### 🚀 [Intermediate Series](intermediate/README.md)
For developers ready to explore more advanced CURSED features.

1. **[Modules and Imports](intermediate/01-modules-imports.md)** - Package system and dependency management
2. **[Generics](intermediate/02-generics.md)** - Generic types and functions
3. **[Interfaces](intermediate/03-interfaces.md)** - Interface definitions and implementations
4. **[Concurrency](intermediate/04-concurrency.md)** - Goroutines, channels, and concurrent programming
5. **[Advanced Pattern Matching](intermediate/05-pattern-matching.md)** - Complex pattern matching scenarios
6. **[Memory Management](intermediate/06-memory-management.md)** - Garbage collection and memory optimization

### 🔬 [Advanced Series](advanced/README.md)
For experienced developers who want to understand CURSED internals.

1. **[Self-Hosting](advanced/01-self-hosting.md)** - Understanding the bootstrap process
2. **[Compiler Internals](advanced/02-compiler-internals.md)** - Lexer, parser, and semantic analysis
3. **[LLVM Codegen](advanced/03-llvm-codegen.md)** - Code generation and optimization
4. **[Writing Stdlib Modules](advanced/04-stdlib-modules.md)** - Creating your own standard library modules
5. **[Performance Optimization](advanced/05-performance-optimization.md)** - Advanced optimization techniques

## Prerequisites

- Basic understanding of programming concepts (helpful but not required for beginners)
- Familiarity with command-line tools
- A working installation of the CURSED compiler

## Learning Path

### If you're new to programming:
Start with the **Beginner Series** and work through each tutorial sequentially.

### If you're experienced with other languages:
You can start with the **Beginner Series** to understand CURSED's unique syntax and then move to the **Intermediate Series**.

### If you're interested in compiler development:
Focus on the **Advanced Series** after completing the foundational tutorials.
